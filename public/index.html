Hello World

This is the creatively named Frontend Template, a simple application template for Frontend Projects.

To get started, just replace the contents of the sass, js and html files - they're only here to display this welcome screen so you can test everything is working ok.

Contributions welcome! Github

Features

  • Webpack & Babel for JavaScript Transforms
  • Gulp for SASS Transforms
  • CSS and JS Source Maps
  • JS checked on comple to ensure it follows JavaScript Standard Style
  • Browser Sync for automatic page reloads
  • Moment.js - Nice Date/Time handling
  • Pre-configured npm scripts. E.g. npm run watch:sass
